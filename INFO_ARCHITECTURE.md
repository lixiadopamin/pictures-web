# 图片工具箱 - 信息架构

## 一、功能模块总览

采用**按功能模块**的信息架构，纯静态托管，无后端依赖。

---

## 二、模块定义（关键词 + 图标）

### 模块 1：图片裁剪

| 项目 | 内容 |
|------|------|
| **图标** | `✂️` 或 剪刀裁剪 (crop) |
| **关键词** | 裁剪、裁切、头像、证件照、自由框选 |
| **功能** | 拖拽裁切区域、缩放、移动、固定比例 |
| **比例预设** | 1:1、3:4、4:3、3:5、5:3、9:16、16:9 |

---

### 模块 2：图片压缩

| 项目 | 内容 |
|------|------|
| **图标** | `🗜️` 或 压缩 (minimize) |
| **关键词** | 压缩、缩小、减体积、优化大小 |
| **功能** | 质量滑块、预览对比、前后大小展示 |
| **支持格式** | PNG、JPG、JPEG |

---

### 模块 3：格式转换

| 项目 | 内容 |
|------|------|
| **图标** | `🔄` 或 循环转换 (repeat/convert) |
| **关键词** | 转换、JPG转PNG、PNG转JPG、格式 |
| **功能** | 选择输出格式，一键转换下载 |
| **支持格式** | JPG ↔ PNG ↔ WebP（可选） |

---

### 模块 4：图片拼图

| 项目 | 内容 |
|------|------|
| **图标** | `🧩` 或 拼图 (layout) |
| **关键词** | 拼图、合成、多图、竖屏、横屏、宫格 |
| **布局类型** | 竖屏、横屏、宫格 |
| **说明** | 常规拼图，非AI，纯前端模板 |

**拼图子类型：**

| 布局 | 图标 | 关键词 | 说明 |
|------|------|--------|------|
| 竖屏 | `📱` | 竖排、上下排列、手机屏 | 多图上下拼接，适应竖屏展示 |
| 横屏 | `🖥️` | 横排、左右排列、宽屏 | 多图左右拼接，适应横屏展示 |
| 宫格 | `⊞` | 九宫格、宫格、网格 | 2×2、3×3、自定义行列 |

---

### 模块 5：通用工具入口（可选）

| 项目 | 内容 |
|------|------|
| **图标** | `🔧` 或 工具 (wrench) |
| **关键词** | 工具箱、常用、快捷 |
| **功能** | 聚合裁剪、压缩、转换的快捷入口 |

---

## 三、导航结构（顶层 Tab / 入口）

```
┌─────────────────────────────────────────────────────────────────────┐
│                        图片工具箱                                     │
├──────────┬──────────┬──────────┬──────────┐
│ ✂️ 裁剪  │ 🗜️ 压缩  │ 🔄 转换  │ 🧩 拼图  │   ← 主入口
└──────────┴──────────┴──────────┴──────────┘
```

或采用**卡片式入口**：

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│  ✂️ 图片裁剪     │  │  🗜️ 图片压缩     │  │  🔄 格式转换     │  │  🧩 图片拼图     │
│  裁切·比例·证件照 │  │  减体积·优化大小  │  │  JPG↔PNG        │  │  竖屏·横屏·宫格  │
└─────────────────┘  └─────────────────┘  └─────────────────┘  └─────────────────┘
```

---

## 四、拼图模块 - 布局选择

```
拼图布局
├── 竖屏 📱
│   └── 多图上下排列，适合朋友圈、短视频封面
├── 横屏 🖥️
│   └── 多图左右排列，适合电脑壁纸、横屏展示
└── 宫格 ⊞
    └── 2×2、3×2、3×3 等网格，九宫格风格
```

---

## 五、技术约束（纯静态）

- 仅使用 HTML5、CSS3、原生 JavaScript
- 不依赖后端、数据库、API
- 部署：GitHub Pages、Vercel、Netlify 等静态托管
- AI Logo 等需 AI API 的功能：暂不纳入，可留作后续扩展
