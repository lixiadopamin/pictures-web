# 图片工具箱

## 预览地址

**在线访问**：<https://your-username.github.io/test/>（部署后请替换为实际地址）

本地预览：浏览器直接打开 `index.html` 或运行 `python -m http.server 8080` 后访问 `http://localhost:8080`

## 项目简介

基于 HTML5 和原生 JavaScript 的**纯静态**在线图片工具箱，覆盖考试报名、日常修图、拼图等场景。部署到 GitHub Pages、Vercel、Netlify 等即可使用。

## 使用说明

### 开发 / 预览（推荐）

```bash
npm install
npm run dev
```

访问 http://localhost:5173，左侧为 Ant Design 导航，右侧为功能内容。

### 静态版本（无构建）

直接打开 `public/legacy.html` 可使用原有卡片式入口；或通过静态服务器（如 `python -m http.server`）访问。

### 上架图生成器（React）

左侧导航点击「上架图生成」进入。功能包括：
- **6 款清淡纹理背景**：可切换，默认第一款
- **样机**：`public/mockup-iphone15.svg` 为默认壳子；可放置 `mockup-iphone15.png` 覆盖
- **主标题 / 副标题**：实时同步到 Canvas 顶部
- **批量导出**：一键生成 1290×2796（6.7 寸）、1242×2208（5.5 寸）
- **Favicon 生成**：导出顶部 Logo 区域为 32×32 PNG（命名 favicon.ico）
- **背景替换**：证件照滤镜，像素操作替换为白/蓝/红 (#FFFFFF / #438EDB / #FF0000)

### 构建部署

```bash
npm run build
```

产物在 `dist/`，可部署到任意静态托管。

## 常见问题

**下载后 Windows 提示「无法打开这些文件」或「Internet 安全设置阻止」**  
- 已改用 Data URL 方式下载，可减轻此问题  
- 若仍出现：右键文件 → 属性 → 勾选「解除锁定」→ 确定
